# poison

> Utilities for writing poisoned types.

## [Documentation](https://crates.fyi/crates/poison/0.1.0)

## Usage

Use the crates.io repository; add this to your `Cargo.toml` along
with the rest of your dependencies:

```toml
[dependencies]
poison = "0.1"
```
